Exercice: Service HTTP

  1. Le fichier /etc/services sert à faire la correspondance entre les noms de services réseau et leurs numéros de ports.
  2. Le numéro du port TCP correspond au port 80.
  3. Le header :
    HTTP/1.0 408 Request Time-out
    Server: AkamaiGHost
    Mime-Version: 1.0
    Date: Mon, 29 Sep 2025 15:56:26 GMT
    Content-Type: text/html
    Content-Length: 314
    Expires: Mon, 29 Sep 2025 15:56:26 GMT
  4. Les 3 valeurs correspond à la version du protocole HTTP, le code de statut 408 et son message associé.
  5. Le Content-Type correspond au type de contenu, ici c'est une page HTML.
  6. Le Content-Length correspond à la taille du contenu, ici il fait 314 octets.
  8. Il est obligatoire de mettre l'Host car on peut avoir plusieurs site en virtuel sur une même IP.
  9. La valeur de location est http://www.igm.univ-mlv.fr/~forax, elle correspond à l'url où le site est hebergé.
