# 📚 ESIEE – Travaux Pratiques

## 📖 Description

Ce dépôt contient l’ensemble des travaux pratiques réalisés durant ma formation à l’ESIEE.

Les projets sont organisés par année et semestre, et couvrent plusieurs domaines de l’informatique : programmation en C, Java, algorithmique, systèmes, bases de données, etc.

## 💻 Contenu

### 🔹 Semestre 1

* **Admin_sys** : notions de systèmes et administration
* **Algo** : algorithmes et structures de base
* **Architecture** : fonctionnement des ordinateurs
* **BDD** : bases de données
* **ProgC** : programmation en C
* **Progweb** : développement web
* **Projet** : projet de semestre

### 🔹 Semestre 2

* **C** : projets avancés en C (ex : Sudoku, Taquin avec MLV)
* **Java** : programmation orientée objet
* **Structure chaînée** : structures de données (listes, etc.)

## ⚙️ Compilation

### 🔸 C (sans MLV)

```bash
gcc fichier.c -o programme
./programme
```

### 🔸 C avec MLV

```bash
gcc fichier.c -lMLV -o programme
./programme
```
